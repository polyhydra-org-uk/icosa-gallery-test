Transformation Descriptions
Dedup
Removes duplicate vertex or texture data, reducing file size without affecting visual quality.
Prune
Removes unreferenced properties and orphaned data from the file.
Quantize
Reduces precision of vertex attributes to decrease file size with minimal visual impact.
Weld
Merges duplicate vertices that share the same position, reducing geometry complexity.
Join
Combines compatible mesh primitives to reduce draw calls and improve performance.
Flatten
Flattens the scene graph hierarchy, which can improve rendering performance.
Instance
Creates GPU instances for reused meshes, significantly reducing file size and improving performance.
Resample
Resamples animation curves to a consistent framerate, potentially reducing animation data size.
Metal-Rough
Converts materials to the metallic-roughness PBR workflow if they use a different workflow.
